# Landing Page for algopulse : Revision Tracker

Developed a personal practice tool. Not currently open for
public signup, this page exists to walk through how it works.

## Usage

```
    Calender 
    Heatmap
    Topic-Wise Progress
    Revision Tracker

```

## Features
• Fetches question details from leetcode while typing in search bar. 
• Heatmap counts one revision = 1 action done and similar for new questions solved.
• Calender shows revision due and question solved on a particular date.
• Topic page shows details abiut topic covered and difficulty of questions solved till date.
• Home page (Dashboard) shows question list due for today or current date.
